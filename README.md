## アプリケーション名
いや！シェア

## アプリケーション概要
イヤイヤ期と言われる2〜3歳前後の子どもを持つ親が、気持ちを共有し励まし合えるポジティブなSNS

## 利用方法
### ポストを投稿する
1.トップページからユーザー新規登録をする
2.ユーザー新規登録後、プロフィール登録を行う
3.投稿フィードの右下にある"投稿する"ボタンから、投稿ページで文章と写真を入力して投稿する

### 他ポストへの共感ボタン（※未実装）
1.投稿フィードから投稿の下部に設置されたボタンでスタンプを選択する
2.投稿の下部にスタンプが反映される

## アプリケーションを作成した背景
自分の子どもが2歳前からイヤイヤ期と言われる、自己主張の時期になった。何を言ってもいや！いや！と言うため、親に余裕がない時は怒ってしまう。
癇癪を恐れたり、癇癪起こした時はどうにもできず、公共の場では周りの視線が痛くて辛いということがある。
SNSで笑いに変えている親をみて、子どもの貴重な時期、このトラブルをもっと楽しく捉えて過ごせたらいいのにと思った。
２歳児のイヤイヤと向き合っている親たちが孤独にならないよう、共感し、励まし、笑いを提供し合いながら繋がれる批判のないSNSを作成しようと思った。

## 工夫したポイント
- SPAによるユーザビリティの向上
  日々忙しいユーザーに待ち時間が発生しないよう、Reactを使ってクライアントサイドルーティングを行なうよう、フロントエンドとバックエンドを分けて作成した。

- 直感的に操作できるデザインを採用
  ユーザーが使い慣れていると思われる一般的なSNSを参考に、操作に迷いを生じにくいデザインと機能とし、何のためのボタンかわかるようにした。

- ポジティブなサービスの提供
　批判がないSNSとして気軽に使用できるよう、コメント欄を設けない、絵文字のみで感情や労りを表せるようなサービス設計にした。（※リアクションボタンは未実装）
